# 使用定时器

1. `[window.]setTimeout()` 延迟定时器
2. `[window.]setInterval()` 循环定时器（间隔器）

- 这两个是 **window 的方法**，所以 this → window 对象，`window.` 可以不写
- 接收 2 个参数：`执行的函数`、`间隔时间`

> ##### 执行的函数

1. 匿名函数

```js
setTimeout(function () {
    console.log('superman');
}, 2000);
```

2. 有名函数

```js
setTimeout(function out() {
    console.log('superman');
}, 2000);
```

3. 有名函数的函数名

```js
function out() {
    console.log('superman');
};
setTimeout(out, 2000);
```

4. 能被当成 JS 语句执行的字符串

```js
setTimeout('console.log("superman")', 2000);
```

5. 箭头函数

```js
setTimeout(() => {
    console.log('superman');
});
```

> ##### 间隔时间，单位：毫秒

```js
setTimeout(() => {
    console.log('superman');
}, 2000);
```

- 省略第 2 个参数会立即执行，间隔时间默认为 4 毫秒

```js
setInterval(function () {
    console.log('superman');
});
```

H5 标准规定 `setTimeout()` 的第二个参数的最小值 >= 4 毫秒。如果 < 4 毫秒，会自动增加。老版本的浏览器都将最短间隔设为 10 毫秒，不同浏览器的实现不同

> ##### 第 3 个及之后的参数：第 1 个函数参数执行的实参

```js
setTimeout((a, b) => {
    console.log(a, b);
}, 2000, 10, 20);
```

> ##### 使用定时器时，会返回该定时器的标识，其实就是定时器的序号

```js
let timer1 = setTimeout(() => {
    console.log('00');
}, 2000);
console.log(timer1) // 1

let timer2 = setTimeout(() => {
    console.log('00');
}, 2000);
console.log(timer2) // 2

let timer3 = setTimeout(() => {
    console.log('00');
}, 2000);
console.log(timer3) // 3
```

> ##### 使用定时器实现圆周运动

- `requestAnimationFrame()` 的使用：能使动画更顺滑，且性能较好

```css
#wrap {
    font-size: 20px;
    font-weight: bolder;
    line-height: 500px;
    position: relative;
    box-sizing: border-box;
    width: 500px;
    height: 500px;
    margin: 50px auto;
    text-align: center;
    border: 1px solid #bbb;
    border-radius: 50%;
}

#wrap span {
    position: absolute;
    top: 0;
    left: 50%;
    width: 10px;
    height: 10px;
    transform: translate(-50%, -50%);
    border-radius: 50%;
    background-color: pink;
}
```

```html
<div id="wrap">
    这是外面的大圆
    <span></span>
</div>
```

```js
let oSpan = document.querySelector("span");
let r = 250; // 半径
let startDeg = 0;

function move () {
    let rad = ++startDeg / 360 * 2 * Math.PI; // 转弧度制
    let x = Math.sin(rad) * r;
    let y = Math.cos(rad) * r;
    let left = x + r;
    let _top = r - y; // top 是保留字，所以使用 _top
    oSpan.style.left = left + 'px';
    oSpan.style.top = _top + 'px';
}

// 使用定时器，不一定能很顺滑
// setInterval(move, 1000 / 60)

function moveSmoothly() {
    move();
    requestAnimationFrame(moveSmoothly); // 递归函数
}

// 递归使用 requestAnimationFrame()，使运动的间隔时间为浏览器的刷新时间，使运动尽可能顺滑
requestAnimationFrame(moveSmoothly)
```



# 清除定时器

使用完定时器后，需要清除定时器。我们可以通过其标识清除定时器

1. `clearTimeout(timer)` → 清除 `setTimeout()`
2. `clearInterval(timer)` → 清除 `setInterval()`

所以，使用定时器时，我们会定义一个变量接收定时器的标识，以便后期删除定时器

```js
let timer1 = setTimeout(() => {
    console.log('00');
}, 2000);
console.log(timer1); // 1
clearTimeout(timer1);

let timer2 = setTimeout(() => {
    console.log('00');
}, 2000);
console.log(timer2); // 2
clearTimeout(timer2);
```

即使前面的定时器被清除了，后面的定时器标识也不会受影响
